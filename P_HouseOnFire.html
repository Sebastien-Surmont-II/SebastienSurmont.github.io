House On Fire!

This game is a Puzzel Platformer Game, where you play as a small flame that needs to make his way to the couch and set it on fire before burning out.

Project Link

The Process

For this project I had the role of team leader, which meant I had to manage a team of 4 while having a tight 3 day deadline. Since it was part of a gamejam.

The biggest problem I ran into in this project was that I had the most experience using the game engine we used (unity). This meant I had to balance managing everyone, scoping the game, planning the features, and help everyone with using the engine.

I learned a lot out of this project like the importance of keeping communication open & applied these things while managing a team for my newest project.

My in game contributions:

SFX & Music, Burning down mechanic, Flame strength mechanic, Death screen, Level setup in unity using art made by artist, Lighting, Respawn point & candle, Making webgl build, Particles

Skills used

Unity, Planning, Scoping, Communication, Managing
